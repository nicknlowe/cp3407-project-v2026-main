# Show current food deals

## Priority: High (10)
This feature will is essential to the app and its competitiveness and thus will make it to the first milestone.

## Assumptions (if any):
* Upon opening the application, FeedMe users will be able to immediately see the latest deals that have been set by partnering businesses
* Users will be able to click/tap on these deals and add them to their cart
* The deals will reset every week or as determined by the partnered businesses
